# MintVue

**MintVue** is a Web3 social NFT platform where creators can **mint viral content as NFTs**, **earn DeFi-based rewards**, and let fans **trade and collect digital moments**. Built on the **TON blockchain**, MintVue combines **NFT creation, social engagement, and yield-linked content** into one seamless experience.
